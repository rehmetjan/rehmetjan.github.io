## 快速开始

### 新建文章

```bash
hexo new "My New Post"
```

了解更多: [写作](https://hexo.io/docs/writing.html)

### 运行服务

``` bash
hexo server
```

了解更多: [服务器](https://hexo.io/docs/server.html)

### 生成静态文件

``` bash
hexo generate
```

了解更多: [生成](https://hexo.io/docs/generating.html)

### 部署到远程机器

``` bash
hexo deploy
```

了解更多: [部署](https://hexo.io/docs/one-command-deployment.html)
